📖 Sobre O projeto Bikcraft foi desenvolvido durante o curso de Web Design Completo da Origamid. Nesse Projeto foi aproveitado a página anterior desenvolvida https://github.com/zGustavo/Bikcraft e transformado ela em um tema WordPress editavel (CMS). 🚀 Tecnologias e ferramentas utilizadas HTML5 CSS3 JavaScript PHP (envio de e-mail com PHPMailer) Wordpress 🖥️ Demonstração 🔧 Como executar o projeto # Clone o repositório git clone https://github.com/zGustavo/Bikcraft-wordpress # Instale o Wordpress # Entre no diretório cd wp-content/themes/bikcraftwp Depois disso, copia o tema para sua pasta de temas dentro do Wordpress.
